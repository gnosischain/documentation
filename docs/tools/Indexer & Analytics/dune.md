# Dune

Dune is the on-chain data platform powering critical business decisions for institutions and enterprises across 100+ blockchains.

## Data Hub

Dune is your onchain data warehouse. Whether you're building dashboards, powering applications, streaming data to your existing warehouse, or running transformation pipelines, any time you need to read aggregated data from the blockchain, Dune can serve it.

## Dune SIM

Sim is a real-time blockchain data API by Dune. It gives developers instant access to wallet balances, token metadata, transaction history, and onchain activity across 60+ EVM chains and Solana — all through a single API key, with no indexer setup required. With webhook subscriptions, you can also push data directly to your app in real time.

## Dune MCP

Dune MCP is Dune's official remote MCP server. It gives your agent direct, structured access to Dune workflows: discover relevant datasets, create and run SQL queries, inspect execution results, manage visualizations, and build dashboards — all in the same conversation.

## Getting Started with Dune

Getting started on Dune is as simple as signing up for an account!

1. Navigate to [dune.xyz](https://dune.xyz) or [sim.dune.com](https://sim.dune.com)
2. Create an account
3. Begin querying
